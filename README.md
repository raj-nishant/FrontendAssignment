# React + TypeScript + Vite

## How to Run

1. Navigate to the frontend directory:

   ```bash
   cd folder_name
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Run the development server:

   ```bash
   npm run dev
   ```

4. Open your browser and visit `http://localhost:5173` to explore the frontend application.

## Testing using Jest

1. Navigate to the frontend directory:

   ```bash
   cd folder_name
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Run test command
   ```bash
   npm test
   ```
